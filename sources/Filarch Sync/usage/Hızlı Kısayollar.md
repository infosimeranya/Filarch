## Hızlı Kısayollar

| Kısayol                                        | İşlem               |
| ---------------------------------------------- | ------------------- |
| `Ctrl + Q` (Windows/Linux) / `Cmd + Q` (macOS) | Uygulamayı Kapat    |
| `F5`                                           | Ekranı Yenile       |
| `Ctrl + ,` (Windows/Linux) / `Cmd + ,` (macOS) | Ayarları Aç         |
| `Tab`                                          | Formda Sonraki Alan |
| `Shift + Tab`                                  | Formda Önceki Alan  |
